URL SHORTENER

User stories:

  • I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  • When I visit that shortened URL, it will redirect me to my original link.

Example usage:

/short?new=http://www.google.com
/short?id=HkhfPjYtM

Example output:

{ "url": http://www.google.com, "short": "http://remigallego.com/demos/api/short?id=HkhfPjYtM" }
